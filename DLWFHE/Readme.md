# 代码说明：

1.enn.py ：基于同态加密的神经网络实现

# 数据说明：

实验中的数据为MNIST中的手写数字图片，每张图片的像素为28*28。实验中的训练集（有标签）和测试集（无标签），以及测试结果均来自Kaggle网站：https://www.kaggle.com/c/digit-recognizer。

1.train.csv ：训练集数据

2.test.csv ：测试集数据
